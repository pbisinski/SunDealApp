# SunDealApp <sup>v0.0.3</sup>
Wychodzimy naprzeciw oczekiwaniom konsumentów i przedstawiamy aplikację typu food sharing. Dziel się, pozyskuj, wymieniaj. 

# O projekcie
Aplikacja pozwala na przeprowadzanie drobnych transakcji handlowych i wymian między użytkownikami. Oddaj lub wymień zbędne produkty. Znajdź je w okolicy dzięki funkcji geolokalizacji. Nie marnuj czasu ani jedzenia! Projekt jest w fazie realizacji, aby sprawdzić ile udało się nam już zrobić sprawdź sekcję <a href="https://github.com/wel-wat/pam-2018-e5t1s1-sundealapp/wiki/Changelog">Changelog</a>!

# O nas
Grupa ambitnych studentów WAT chcących zaliczyć projekt.
